    Service Agent Console View

    1. Go to app picker and select a console app,it will head to a page where agent can chat with customer.
    2. Make sure the agent is not in offline, agent should be active.

    1. If customer needs any attention, bot will connect chat to agent, where a modal window pops up showing the request from bot to connect to customer.

    1. Hovering on accept button will show the conversation between chatbot and customer.
    2. By clickig on accept the agent can start chatting with the customer.
    3. From here the agent can able to create a lead or a case based on the information collected.
